Merge sort

Time complexity

best/worst/average: nlogn

Space complexity

n

#Steps:
1. find the mid l + (r-l)/2
2. Call merge sort for each half
 mergeSort(arr,l,m);
 mergeSort(arr,m+1,r);
3. merge sorted halves
